# GraphQL
> [Official Site](https://graphql.cn/)  Graph Query Language 简称 GQL 面向图结构的查询语言

相较于传统的API，此方案创造性了提出 要什么就给什么 将接口的主导方从提供方转到调用方，优劣也是很明显，因此争议也很多，酌情按需使用。
