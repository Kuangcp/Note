# SVG
> [MDN:SVG ](https://developer.mozilla.org/zh-CN/docs/Web/SVG)

> 可缩放矢量图形（Scalable Vector Graphics，SVG)，是一种用来描述二维矢量图形的 XML 标记语言。 简单地说，SVG 面向图形，HTML 面向文本。
