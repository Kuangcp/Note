# WSL2
- `启动发行版` wsl -d Name

> 更换磁盘地址

- 导出已有发行版 `wsl --export Ubuntu D:/wsl/Ubuntu-disk.tar` 
- 导入成为新的发行版并设置数据存放目录 `wsl --import Ubuntu2 D:/wsl/Ubuntu2 D:/wsl/Ubuntu-disk.tar`
- 删除旧发行版 `wsl --unregister Ubuntu`
